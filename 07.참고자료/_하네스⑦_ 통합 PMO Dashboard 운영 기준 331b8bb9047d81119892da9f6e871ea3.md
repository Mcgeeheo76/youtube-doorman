# 통합 PMO Dashboard 운영 기준

## 핵심 요약
6개 하네스 프로젝트를 단일 화면에서 실시간 모니터링하기 위한 통합 PMO Dashboard 운영 기준 문서이다. Notion TASKS DB와 AX PROJECTS DB를 연동하여 Gate 상태, KPI 트렌드, 리스크 Heatmap을 주간·월간·분기 주기로 관리한다. 현재 가장 높은 리스크는 34억 예산 불일치로 KICOX 협의가 필요하며, 설계-시공 불일치 및 협력기관 지연도 주요 리스크로 분류된다. 담당자 허민영이 Dashboard 전반의 업데이트와 보고자료 생성을 총괄한다.

## 주요 내용
- 6개 하네스(설계관리·GMP리모델링·장비구축·리트로핏·PMO Core·외부기관)의 Gate 진행 현황을 1페이지에서 통합 모니터링
- 리스크 Heatmap 최고 위험 항목은 34억 예산 불일치로, 확률·영향도 모두 최고 등급(🔴🔴🔴)
- 즉시 실행 우선순위는 예산 불일치 KICOX 협의, 발주 크리티컬패스 확정, TASKS DB 연계, IQ/OQ/PQ 템플릿 제작 순
