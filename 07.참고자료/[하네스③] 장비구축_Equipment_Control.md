## 📋 장비 A · GPU 서버 체크리스트

### URS (User Requirement)

- [ ]  GPU 사양 요구사항 정의 (VRAM, 연산 성능)
- [ ]  원격접속 환경 요구사항 정의 (홍 차장님 협의)
- [ ]  냉각 및 전력 용량 요구사항 확인

### FAT (Factory Acceptance Test)

- [ ]  제조사 공장 테스트 결과 수령
- [ ]  성능 기준 vs 실측값 대조 확인
- [ ]  FAT 체크리스트 서명 완료

### SAT (Site Acceptance Test)

- [ ]  현장 설치 후 성능 재확인
- [ ]  원격접속 연결 테스트 완료
- [ ]  SAT 체크리스트 서명 완료

### IQ/OQ/PQ

- [ ]  IQ: 설치 위치·전원·네트워크 적합성 확인
- [ ]  OQ: 운영 파라미터 범위 테스트
- [ ]  PQ: 실제 AI 학습/추론 부하 조건에서 안정성 확인
